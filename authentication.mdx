## Overview

Secure authentication protects your patient management system at wartenmitadana GmbH. You access the dashboard to handle patient queues, staff assignments, and call notifications. All logins use HTTPS encryption, and sessions expire automatically for safety.

<Callout kind="info">
  Before logging in, ensure you have an active account. New users register at `https://app.example.com/register` with practice details verified by email.
</Callout>

## API Authentication

Integrate with your systems using token-based auth. First, generate an API token.

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token: `Bearer YOUR_TOKEN`.
</ParamField>

<ParamField query="practice_id" param-type="string" required="true">
  Your practice identifier.
</ParamField>

<Request tabs="cURL,JavaScript">
  ```bash
  curl -X POST https://api.example.com/v1/auth/login \
    -H "Content-Type: application/json" \
    -d '{
      "email": "user@example.com",
      "password": "securepass123"
    }'
  ```
  ````javascript
  const response = await fetch('https://api.example.com/v1/auth/login', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      email: 'user@example.com',
      password: 'securepass123'
    })
  });
  const data = await response.json();
  console.log(data.token);
  ````
</Request>

<Response tabs="200,401">
  ```json
  {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "expires_at": "2024-12-31T23:59:59Z"
  }
  ```

  ```json
  {
    "error": "Invalid credentials"
  }
  ```
</Response>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const token = data.token;
  fetch('https://api.example.com/v1/patients', {
    headers: { Authorization: `Bearer ${token}` }
  });
  ```
  ```python
  import requests
  headers = {'Authorization': f'Bearer {token}'}
  response = requests.get('https://api.example.com/v1/patients', headers=headers)
  ```
</CodeGroup>

## Session Handling and Logout

Sessions last 30 minutes of inactivity before auto-logout. Manually log out from the profile menu.

<Callout kind="alert">
  Always log out from shared devices. Never share credentials or tokens.
</Callout>
